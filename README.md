Welcome to the repository for my personal portfolio website! This project serves as a digital resume and a central hub to showcase my skills, educational background, professional experience, and the software projects I've built.

## ✨ Features

- **Modern & Responsive UI:** A clean, aesthetically pleasing design featuring glassmorphism elements, smooth transitions, and responsive layout for all devices.
- **Interactive Animations:** Engaging typing effects, scroll reveals, and a dynamic splash screen.
- **Projects Showcase:** A dedicated section highlighting my best work, including live demos and GitHub source links.
- **Downloadable Resume:** Quick access for recruiters to download my latest PDF resume.

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Icons & Fonts:** FontAwesome, Google Fonts (Poppins)
- **Deployment:** Vercel

## 💻 Getting Started

To run this project locally, you simply need to clone the repository and open the `index.html` file in any modern web browser. No complex build tools or package managers are required.

```bash
# Clone the repository
git clone https://github.com/srivatsan2007/portfolio.git

# Navigate to the project directory
cd portfolio

# Open the project (Windows)
start index.html
```
